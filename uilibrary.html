Welcome to the LifeasHub UI Library

This document is designed to help you understand all the features and usage of the LifeasHub UI library. With its modern, flexible, and customizable structure, you can easily create stylish interfaces for your projects.

The library is built on a tabbed structure and offers many ready-to-use elements such as buttons, sliders, and dropdown menus. It also comes with advanced features like theme management and a modern notification system. You can find more info at our official website.

Setup

Adding the library to your project is very simple. Just paste the following code into a LocalScript (e.g., in StarterPlayerScripts).

-- Load the library from its source
local LifeasHubLibrary = loadstring(game:HttpGet("https://raw.githubusercontent.com/barann222/projectnexus/refs/heads/main/uilibrary"))()

Initializing the Library Library:Init

The Library:Init() function is used to create the main window of the interface. This function takes a table containing the basic settings for the window and returns a Window object that you can work with.

local Window = LifeasHubLibrary:Init({
    name = "PROJECT NAME",
    name2 = "LifeasHub",
    name3 = "v1.0 | by Your Name",
    defaultTheme = "Blurple",
    defaultThemeLevel = 1
})

Parameters

ParameterTypeDescription
namestringThe main title in the top-left corner of the window.
name2stringThe second, colored title that appears next to the main title.
name3stringA subtitle that appears to the right of the titles (e.g., version).
defaultThemestringThe initial theme. E.g., 'Red', 'Green', 'Blurple'.
defaultThemeLevelintegerBackground theme intensity (0: Off, 1: Standard, 2: Intense).

Tabs Window:CreateTab

All interface elements are contained within tabs. You can create a new tab using the Window:CreateTab() function.

local MainTab = Window:CreateTab({ name = 'Main' })
local SettingsTab = Window:CreateTab({ name = 'Settings' })

-- This makes the tab visible when the UI is first opened
MainTab:Activate()

Methods

Tab:Activate(): Makes the corresponding tab visible and active. You should use this method to ensure a tab is displayed when the UI first loads.

Elements

The library provides various elements to build your interface. All elements are created from a Tab object.

Button

A standard button that performs an action when clicked.

MainTab:Button({
    name = "Send Notification",
    callback = function()
        Window:Notify({title="Success!", content="The button works!", type="success"})
    end
})

Toggle

A switch that can be turned on or off.

SettingsTab:Toggle({
    title = "Show ESP",
    callback = function(isEnabled)
        print("ESP status: ", isEnabled)
    end
})

Slider

Used to select a numerical value within a specific range.

SettingsTab:Slider({
    title = "WalkSpeed",
    min = 16,
    max = 100,
    default = 16,
    callback = function(value)
        print("New walk speed: ", value)
    end
})

Dropdown

A dropdown menu used for single or multiple selections from a list.

SettingsTab:Dropdown({
    title = "Target Part",
    items = {"Head", "Torso", "Legs"},
    multiSelect = false,
    callback = function(selected)
        print("Selected target: ", selected)
    end
})

TextInput

A box for getting text input from the user.

MainTab:TextInput({
    title = "Custom Name",
    placeholder = "Enter your new name...",
    callback = function(text)
        Window:Notify({title="Name Changed", content = "Your new name is: " .. text})
    end
})

Label

Used to separate groups of elements or to add a heading to a section.

SettingsTab:Label({message = "General Settings"})

Paragraph

An informational box containing a title, a long description, and an optional image.

MainTab:Paragraph({
    title = "About The Project",
    content = "This UI was designed to provide the best user experience.",
    image = "rbxassetid://6019884959"
})

Advanced Features

Powerful features offered by the library beyond the basic elements.

Notification System Window:Notify

Displays modern, animated notifications in the top-right corner of the screen.

-- Notification with an icon
Window:Notify({
    title = "Warning!",
    content = "You are about to reset the settings.",
    type = "warning",
    duration = 4
})

-- Notification with a banner image
Window:Notify({
    title = "Update Available!",
    content = "Version 2.0 has been released.",
    image = "rbxassetid://2853763354",
    imageStyle = "banner"
})

Parameters

ParameterTypeDescription
titlestringThe title of the notification.
contentstringThe content of the notification.
durationnumberHow long it stays on screen (seconds). Default: 5.
typestringPreset icon type: 'success', 'warning', 'error', 'info'.
imagestringAn rbxassetid for a custom image.
imageStylestringImage style: 'icon' (default) or 'banner'.

Theme Management

You can change the colors and theme of the interface in real-time.

Window:SetTheme(themeName): Changes the accent color of the interface.

Window:SetBackgroundThemeLevel(level): Sets how much the background is affected by the theme.

Full Example

Here is a complete example script demonstrating many features of the library in action. You can copy and paste this into a LocalScript to see how everything works together.

-- 1. Load the library
local LifeasHubLibrary = loadstring(game:HttpGet("https://raw.githubusercontent.com/barann222/projectnexus/refs/heads/main/uilibrary"))()

-- 2. Initialize the main window
local Window = LifeasHubLibrary:Init({
    name = "LifeasHub DEMO",
    name2 = "v2.1",
    name3 = "by LifeasHub Team",
    defaultTheme = "Green",
    defaultThemeLevel = 1
})

-- 3. Create tabs
local MainTab = Window:CreateTab({ name = 'Showcase' })
local SettingsTab = Window:CreateTab({ name = 'Options' })

-- 4. Populate the MainTab with elements
MainTab:Paragraph({
    title = "Welcome to the Showcase!",
    content = "This tab demonstrates various notification types and user input elements. Click the buttons below to see different notifications.",
    image = "rbxassetid://6019884959" -- Example Image
})

MainTab:Label({message = "Notification Tests"})

MainTab:Button({
    name = "Success Notification",
    callback = function()
        Window:Notify({title="Success", content="The operation was completed successfully.", type="success"})
    end
})

MainTab:Button({
    name = "Error Notification (Custom Duration)",
    callback = function()
        Window:Notify({title="Error!", content="Failed to connect to the server.", type="error", duration = 8})
    end
})

MainTab:Button({
    name = "Warning Notification",
    callback = function()
        Window:Notify({title="Warning", content="Your settings are not saved.", type="warning"})
    end
})

MainTab:Button({
    name = "Info Notification with Banner",
    callback = function()
        Window:Notify({title="Did you know?", content="LifeasHub UI supports banner images.", type="info", image = "rbxassetid://2853763354", imageStyle = "banner"})
    end
})

MainTab:Label({message = "Input Example"})

MainTab:TextInput({
    title = "Enter a message",
    placeholder = "Type something here...",
    callback = function(text)
        if text and #text > 0 then
            Window:Notify({title="Message Sent", content = "You entered: " .. text, type="info"})
        else
            Window:Notify({title="Oops!", content = "You didn't enter anything.", type="warning"})
        end
    end
})


-- 5. Populate the SettingsTab with elements
SettingsTab:Label({message = "Visual Settings"})

SettingsTab:Toggle({
    title = "Enable Glow Effect",
    callback = function(isEnabled)
        print("Glow effect is now:", isEnabled)
        Window:Notify({title="Settings Updated", content="Glow effect set to " .. tostring(isEnabled), type="info"})
    end
})

SettingsTab:Slider({
    title = "Field of View",
    min = 70,
    max = 120,
    default = 80,
    callback = function(value)
        print("FOV set to: ", value)
        -- You could set game.Workspace.CurrentCamera.FieldOfView = value here
    end
})

SettingsTab:Label({message = "Selection Controls"})

SettingsTab:Dropdown({
    title = "Aim Target (Single)",
    items = {"Head", "Torso", "Right Arm", "Left Arm"},
    multiSelect = false,
    callback = function(selectedItem)
        print("Single selection:", selectedItem)
    end
})

SettingsTab:Dropdown({
    title = "ESP Features (Multiple)",
    items = {"Boxes", "Names", "Tracers", "Health"},
    multiSelect = true,
    callback = function(selectedItems)
        print("Multi-selection changed. Current active features:")
        for item, isSelected in pairs(selectedItems) do
            if isSelected then
                print("- " .. item)
            end
        end
    end
})

SettingsTab:Label({message = "Theme Controls"})

SettingsTab:Dropdown({
    title = "Accent Color",
    items = {"Red", "Green", "Blue", "Yellow", "Pink", "Orange", "Purple", "Blurple"},
    multiSelect = false,
    callback = function(selectedTheme)
        Window:SetTheme(selectedTheme)
        Window:Notify({
            title = "Theme Changed", 
            content = "Accent color set to " .. selectedTheme, 
            type = "info", 
            duration = 3
        })
    end
})

SettingsTab:Slider({
    title = "Background Intensity",
    min = 0,
    max = 2,
    default = 1,
    callback = function(value)
        local intValue = math.floor(value + 0.5) -- Round to nearest integer
        Window:SetBackgroundThemeLevel(intValue)
    end
})


-- 6. Activate the first tab to make it visible
MainTab:Activate()
